# Tshirt Shop

This is a project the was built for Turing's Fulstack challenge
This web app is built in a docker environment and it is a ecommerce store
that sells tshirts.

Features:
<ul>
<li>Authentication</li>
<li>Pagination</li>
<li>Shopping Cart</li>
<li>Search Engine</li>
</ul>
